# 供需字段模板

## 1. 算力资源发布模板

### 基础信息

- 资源标题
- 资源类型
- 企业所在地
- 可用时间
- 最短租期
- 是否可长期合作
- 是否支持现场考察

### 硬件信息

- GPU 型号
- GPU 数量
- 单卡显存
- CPU 型号
- 内存容量
- 本地存储
- 共享存储
- 网络类型
- 网络带宽
- 是否支持 RDMA
- 是否支持 InfiniBand

### 服务信息

- 交付方式
- 是否支持裸金属
- 是否支持容器
- 是否支持私有化部署
- 是否提供运维
- 是否支持专线
- 是否支持公网访问
- 是否支持数据迁移

### 价格信息

- 计费方式
- 价格区间
- 是否含税
- 付款方式
- 押金要求
- 是否可议价

### 资质与证明

- 企业资质
- 资源证明
- 机房证明
- 网络证明
- 历史案例
- 测试报告

## 2. 算力需求发布模板

### 基础信息

- 需求标题
- 企业类型
- 企业所在地
- 使用场景
- 期望开始时间
- 预计使用周期
- 预算范围

### 技术需求

- 训练或推理
- GPU 型号偏好
- GPU 数量
- 显存要求
- 存储要求
- 网络要求
- 是否需要 RDMA
- 是否需要国产化
- 是否需要裸金属
- 是否需要运维支持

### 合规需求

- 数据是否敏感
- 是否要求本地部署
- 是否要求等保
- 是否要求信创
- 是否限制地域
- 是否需要签署保密协议

### 商务需求

- 期望付款方式
- 是否需要发票
- 是否接受代理商
- 是否需要平台撮合
- 是否希望线下见面

- 单柜功率

- PUE

- 运营商线路
- 是否支持双线或多线
- 是否支持专线
- 是否支持液冷
- 是否支持高密机柜
- 交付周期
- 托管价格
- 增值服务

- 总电力容量
- 是否需要液冷

- 是否需要施工总包

- 是否需要运维系统
- 预算范围
