# 算力贸易小程序 · UI 页面规范（v1.0）

> 与 `app.wxss` 全局样式配套。新页面请按本规范组织 WXML，保证风格统一、结构清晰（MECE）。

---

## 一、设计原则

1. **一套 Token**：颜色、圆角、间距以 `app.wxss` 为准，页面 WXSS 只写本页特例。
2. **MECE 分块**：每页按「页头 → 提醒 → 工具栏 → 内容区 → 底栏」划分，块与块互斥、合起来覆盖整页。
3. **标题三级**：页级 `section-title`、卡片内 `section-title compact`、Tab 子模块 `section-title subtle`。
4. **空态一种**：列表无数据统一 `.empty`；表单内局部无选项用 `.empty.empty-inline`。

---

## 二、全局类名（必用）

| 类名 | 用途 |
| --- | --- |
| `.page` | 所有页面根容器 |
| `.page-tab` | 底部 Tab 页（首页/资源/需求/我的） |
| `.page-block` | MECE 内容模块外层 |
| `.section-title` | 页级区块标题（32rpx） |
| `.section-title.compact` | 卡片内一级标题（30rpx） |
| `.section-title.subtle` | Tab 页柔和标题（24rpx，配 `.section-desc`） |
| `.section-desc` | 模块说明一行 |
| `.card` | 白底内容卡片 |
| `.empty` | 列表空状态 |
| `.button-primary` / `.button-secondary` | 主/次按钮 |
| `.fixed-bottom-bar` + `.has-fixed-bar` | 固定底栏与页面留白 |
| `.hero-card` | 深色页头（记录/审核/流程） |
| `.res-section` + `.res-section-head` | 商机详情区块 |
| `.form-section` | 表单卡片内分组标题 |

主色：`#176b5b` · 正文：`#17212b` · 次要：`#68717d` · 背景：`#f5f7fa`

## 三、页面 MECE 结构模板

```
.page [.page-tab] [.has-fixed-bar]
├── 页头（hero-card / user-header / pool-v2-toolbar）
├── 提醒（limited-banner / next-step / cloud-bar）
├── 工具栏（search-box / filter-row / admin-tabs）  ← 列表/运营页
├── 内容区（page-block 可重复）
│   ├── .section-title [.subtle] + .section-desc
│   └── .card | .pool-card-v2 | .menu-group
└── .fixed-bottom-bar（可选）
```
